Working Projects

DrawTogether

DrawTogether is my personal favourite project. It is node.js based website that allows users to collaborate on drawings in real time. It also allows them to stream their drawing to spectators. To put it simply, it is more or less multiplayer ms paint.

2.5D

2.5D is a game with its own orthogonal engine written in vanilla js. It uses perspective to allow you to play 3 different 2 dimensional minigames in multiplayer in 3d space.

2.5D on Github

Cyberstick

Cyberstick is a website that allows you to instantly and anonymously share files. It is made especially for journalists / persons of interest in countries with very strict and punishing regimes.
